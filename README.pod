=head1 NAME

Nephia::View::TT - Template for Nephia

=head1 SYNOPSIS

  ### in your app class ...
  package MyApp;
  use Nephia;
  
  ### and your app.psgi ...
  use MyApp;
  MyApp->run( view => {
      class => 'TT',
      TEMPLATE_PATH => [ ... ],
      EVAL_PERL => 1,
  } );

=head1 DESCRIPTION

Nephia::View::TT is view class for Nephia. It provides template feature as Template.pm.

=head1 USING IN SETUP

  nephia-setup Your::AppName --flavor View::TT
